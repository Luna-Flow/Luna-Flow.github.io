# float_backend 設計

`float_backend` パッケージは、複素数の浮動小数点解析挙動を担当します。

## 責務

- 浮動小数点演算、特殊値、分岐切断のセマンティクスをルートパッケージから切り離す。
- より広いエコシステム trait が必要になるまでは、バックエンド能力 trait をこのパッケージ内に保つ。
- 現在の `Complex[Double]` の安定性挙動を維持しつつ、将来の `Float` 解析関数が同じバックエンド能力面を共有できるようにする。
- helper は原則として内部に保ち、文書化された公開 API だけをエクスポートする。

## 保守メモ

- 公開バックエンド trait や解析関数シグネチャが変わったときは、このページを更新する。
- `Complex[Float]` の解析関数が実際にエクスポートされるまでは、完全対応済みであるかのように書かない。
