# core 設計

ルートパッケージは、汎用 `Complex[T]` 構造、代数演算、trait による制約を担当します。

## 責務

- `Complex[T]` を汎用のまま保ち、浮動小数点の解析セマンティクスを入れない。
- 呼び出し側が `re` や `im` を意図的に更新できるよう、可変フィールドモデルを維持する。
- 代数 trait 実装は、実際に必要な最小のスカラー能力だけを要求する。
- 分岐切断、NaN、無限大、負のゼロ、超越関数は `float_backend` に置く。

## 保守メモ

- ルートパッケージの公開 trait 実装が増減したときは、このページを更新する。
- 未実装の汎用解析 API をルートパッケージの文書に書かない。
