# float_backend 设计

`float_backend` 包负责复数的浮点解析行为。

## 职责

- 将浮点算术、特殊值和分支切割语义隔离在根包之外。
- 后端能力 trait 暂时保留在本包内，直到生态层面确实需要更通用的 trait。
- 保持当前 `Complex[Double]` 的稳定性行为，同时为未来 `Float` 解析函数复用同一后端能力面预留空间。
- helper 默认保持内部可见；只有明确属于公开 API 的函数才导出。

## 维护说明

- 公开后端 trait 或解析函数签名变化时，同步更新本文档。
- 在 `Complex[Float]` 解析函数真正导出之前，不要把文档写成已经完整支持。
