# Luna-Flow/luna_thread

本文档跟踪当前分支状态。

## 仓库定位

带有运行时后端与规范资产的 MoonBit 并行 FFI 规范工作区。

## 文档布局

- `README.md` 记录仓库叙事与版本基线。
- `doc_standard.md` 记录文档契约。
- 模块或子系统目录下放 `api.md`、`tutorial.md`、`design.md`。

## 模块概览

- **`facade`**: 主要实现位于 `luna_thread`。
- **`backend`**: 主要实现位于 `native and js`。
- **`spec`**: 主要实现位于 `docs`。
